# Juego de damas
## Descripcion
Juego de damas por consola
- Programacion orientada a objetos

## Compilar
```bash
ant
```
## Ejecutar
```bash
java -jar ./dist/lib/JuegoDamas.jar
```
## Nota
Al ejecutar elegir el .jar que se creo despues de la compilacion.
